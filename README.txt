This is a Java Spring Boot web application. It calculates delivery fees for couriers depending on the city and vehicle selected. 
The fee also depends on the current weather condition in that city(gathers that data from a public database).
To access the interface, run CourierApplication.java and visit page http://localhost:(closest open port number)/interface.html#
To access the database, run CourierApplication.java and visit page http://localhost:(closest open port number)/h2-console/ and click connect
PS! The username is "sa" and password an empty string(default settings).
